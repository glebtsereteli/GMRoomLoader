
# GMRoomLoader v3.1.0 Demo

## Welcome!

Welcome to the [GMRoomLoader](https://github.com/glebtsereteli/GMRoomLoader) demo project!

This project showcases the core functionality of the library. It's designed as a feature demonstration and reference, not a tutorial. For a guided introduction, check out the [First Setup](https://glebtsereteli.github.io/GMRoomLoader/pages/home/demosTutorials/demosTutorials#first-setup) project and the available [Tutorials](https://glebtsereteli.github.io/GMRoomLoader/pages/home/demosTutorials/demosTutorials#tutorials).

This project can also be played in the browser on the [itch page](https://glebtsereteli.itch.io/gmroomloader).

## What's Included

The demo project includes interactive examples covering most features of the library.

It also features a DBG ImGui control panel that allows you to cycle through demos and access custom controls for each example.

Learn more about the demo in the [Demo Documentation](https://glebtsereteli.github.io/GMRoomLoader/pages/home/demosTutorials/demosTutorials#demo-project).
