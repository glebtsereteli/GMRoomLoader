
# Welcome

Welcome to GMRoomLoader v3.1.0 First Setup!

This project is a part of the First Setup documentation page you can find here: https://glebtsereteli.github.io/GMRoomLoader/pages/home/gettingStarted/gettingStarted#first-setup

For a more complete picture of what GMRoomLoader can do, check out the [Demo Project](https://glebtsereteli.github.io/GMRoomLoader/pages/home/demosTutorials/demosTutorials#demo-project). It can be downloaded or played in the browser on the [itch page](https://glebtsereteli.itch.io/gmroomloader).

# Instructions

- `objHost` handles room loading and [Payload](https://glebtsereteli.github.io/GMRoomLoader/pages/api/payload/overview) cleanup. Relevant events: Create, Step.
- `rmExample` is the example room we're loading. It has instances, tiles, and sprite assets.
- Press `1` to load the room.
- Press `2` to unload the room.
